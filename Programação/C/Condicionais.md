## If
```C
int i;
scanf("%i",&i);
if (i < 12) // o código só vai rodar se a condição (i for menor que 12) for verdadeira
{
	printf("i é menor que 12\n");
}
```

## Elif
```C
int i;
scanf("%i",&i);
if (i < 12)
{
	printf("i é menor que 12\n");
}
else if (i < 24) // pergunta caso a primeira seja falsa
{
	printf("i é maior que 12, porém menor que 24\n");
}
```

## Else
```C
int i
scanf("%i",&i)
if (i < 12)
{
	printf("i é menor que 12\n");
};
else // roda sempre que o if correspondente for falso
{
	printf("i é maior ou igual a 12\n"); 
}
```
